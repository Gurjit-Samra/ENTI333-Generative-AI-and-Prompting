# ENTI 333/633 – Generative AI and Prompting

This repository contains course deliverables for **ENTI 333 / ENTI 633: Generative AI and Prompting**.

## Course Description
Learn to use generative AI effectively by understanding the technology, and using prompting techniques, templates, and tools. The course emphasizes innovative and entrepreneurial applications through direct use of generative AI tools and models, as well as indirect use through APIs. It also involves scripting and no-code tools to build practical generative AI solutions.

> Note: ENTI 633 is merged with the undergraduate course (ENTI 333), but this outline is for the graduate version. Graduate students are held to higher expectations for select grading components.

## Course Modality
- Primarily **in-person**.
- Some components (e.g., lab exercises and group project work) may not require strict in-class presence at all times.
- There are no fully online or asynchronous lecture components.
- The instructor may provide audio recordings of lectures after delivery.

## Course Objectives
By the end of the course, students will be able to:

1. Understand how generative AI works and identify relevant tools, trends, and societal implications.
2. Demonstrate how generative AI knowledge is applied through prompting.
3. Understand, analyze, and discuss ethical issues around generative AI.
4. Apply generative AI tools directly to design solutions and create value.
5. Apply generative AI tools indirectly through APIs and scripting to design solutions and create value.
6. Identify problems best suited for generative AI and cases where it may be prudent not to use it.
